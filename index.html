  1. Install XAMPP
  2. Pastikan XAMPP anda sudah berjalan dengan cara, mengetikkan http://localhost di web browser anda
  3. Buat folder baru di folder xampp/htdocs
  4. buat file index.html, silahkan edit dengan sublime text editor
> Cara Kerja Codeigniter
  1. pada saat membuka halaman browser dengan folder evaluasi, maka yang pertama kali akan diakses adalah folder controller dengan nama Welcome. Kemudian fungsi yang pertama kali diakses secara otomatis adalah fungsi index(). Hal ini dikarenakan configiurasi routes secara default diarahkan ke controller welcome
  2. dari fungsi index(), membuka file di folder view dengan nama welcome_message
Tugas
  1. buat controller baru dengan nama C_MyWelcome
  2. buat fungsi index() yang membuka file di folder view dengan nama v_mywelcome. Dengan catatan v_mywelcome.php sudah dibuat terlebih dahulu
  3. Silahkan halamannya di refresh
